Help Area

Timer

When the committee sets a Speech and Comment time, enter them into the respective Speech and Comment text box. To load either timer into the main top timer, click the Speech button or Comment button, which will load the time. Proceed to click "Start" to start the time or "Pause" to pause it. The Caucus timer is controlled from the two buttons under it. Simply enter a time and click "Start" to start or "Pause" to pause. The Caucus timer is used for both unmoderated and moderated caucuses; in addition, during a moderated caucus, the Caucus timer and Main timer can be run at the same time. The timer boxes will automatically format to mm:ss so it is necessary to type each minute and second (for example a 15 second comment time would be entered as 00:15). When switching from Speeches to Comments or visa versa, click "Pause" to give the delegate time to stand and begin. When deleting the times, it is easiest to hold the delete key down to delete and replace them.

Speakers' List

Make sure that each country name in the Speakers' List matches the listed name in the dropdown menu. It is recommended to select the country from the menu, press Enter, then press Tab to continue entering in the country names. Once all the speakers have been entered, click the "Next Speaker" button at the base of the Speakers' List. The current speaker will be highlighted in yellow and the previous speakers will have a strike-through. If there are two delegates of the same country, simply type their country name and a number separated by a space (for example: "France 1"). If more Country boxes are required, scroll to the bottom of the Speakers' List and click the "More Speakers" button. If multiple Speakers' Lists are required for Moderated Caucus for example, type "Speakers' List 2" in the Speakers' List box then add the speakers underneath and click "Next Speaker".

Options Page

Right click on the extension's logo to access the options page. There you are able to choose a from dropdown menu of premade options or customize your own. Reload page to see the effect(s).
